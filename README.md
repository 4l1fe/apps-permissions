Для запуска необходимо скачать репозиторий как есть. Иметь установленный docker/docker-compose в системе.
Выполнить команду из корня репозитория `docker compose up -d`, по ней произойдет запуск трех контейнеров, один из которых имеет отложенный запуск(создание БД и таблицы)


Далее обратиться в веб интерфейс по адресу `127.0.0.1:8080` и ввести ссылку вида `https://play.google.com/store/apps/details?id=com.mobile.legends&hl=en`.
